# **App Name**: EduQuest

## Core Features:

- Interactive Learning Modules: Engaging modules for Maths, English, and Science with games, exercises, and quizzes tailored for children aged 6-12.
- Gamified Progress Tracking: Level system, badges, and points to motivate children, tracking their performance over time.
- User Accounts & Profiles: Secure signup/login with profile customization and progress tracking.
- Personalized Learning Paths: AI-powered feature to analyze child's performance and customize learning path, by choosing modules using an analysis tool for current knowledge, skills, and knowledge gaps.
- Quiz Generation: Create customized quizzes based on the modules learned, using generative AI. AI model will choose questions that cover gaps from previously completed quizzes using a quiz generation tool.

## Style Guidelines:

- Primary color: Cheerful blue (#42A5F5) to evoke a sense of trust and learning.
- Background color: Soft, desaturated blue (#E3F2FD) for a calm and inviting learning environment.
- Accent color: Bright orange (#FF9800) to highlight important interactive elements and CTAs.
- Body and headline font: 'PT Sans', a humanist sans-serif suitable for both headlines and body text, ensuring readability for children.
- Use bright, friendly, and illustrative icons for easy navigation.
- A grid-based layout for clear organization, child-friendly navigation with large buttons.
- Subtle, playful animations and transitions for engaging interactions.
